# Sistema de Gestão de Funcionários - Java Collections 🚀

Este projeto foi desenvolvido por mim como um exercício prático de fixação dos conceitos de Java.

## Créditos
Este exercício faz parte do curso ministrado pelo professor **Nelio Alves**.

## Conceitos Aplicados
* **Encapsulamento:** Uso de modificadores de acesso, getters e setters.
* **Java Streams & Lambda:** Uso de predicados para filtragem e busca eficiente de dados em listas (métodos `filter` e `findFirst`).
* **Comportamento de Memória:** Diferenciação entre tipos primitivos e tipos Wrappers (`Integer`, `Double`).
* **Lógica de Programação:** Validação de unicidade de ID e manipulação de listas dinâmicas.

## Como funciona
O programa permite registrar um número 'N' de funcionários, garantindo que não haja IDs duplicados durante a inserção. Após o registro, é possível buscar um funcionário pelo ID e aplicar um aumento salarial baseado em uma porcentagem informada.
